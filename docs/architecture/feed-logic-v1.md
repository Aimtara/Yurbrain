# Feed Logic v1

The Focus feed is a curated, stateful, ranked memory surface.

It is not:
- an inbox
- a task list
- a timeline
- an engagement stream

Card types:
- item
- digest
- cluster
- opportunity
- open_loop
- resume

Lenses:
- all
- keep_in_mind
- open_loops
- learning
- in_progress
- recently_commented

Core rules:
- selective, not exhaustive
- reversible
- why-shown always available
- diversity to avoid topic flooding
